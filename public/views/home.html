Demo Api Rest By DevApp

- This is a simple CRUD for articles with AngularJs And ExpressJs

dans cette petite demo nous allons utiliser le framework ExpressJs partie Back-end et AngularJs-1.x Partie Frent-end avec bootstrap et Material design lite coté designe css3 et en fin MongoDB comme DATABASE. Tout ça fonctionne sous le plateforme Node Js NodeJs

dans cette application on peut ajouter un produit a partir d'une simple formulaire sité dans la section "Articles" clicker sur le lien dans le menu si dessus , et a partir de la page articles vous pouvez ajouter , modifier ou supprimer les articles
